# Skill: search-articles
Запускает еженедельный поиск статей по текущей теме. Вызов: `/search-articles`

Используй агентов `researcher` → `deduplicator`.

## Перед запуском:
1. Прочитай `topics/current_topic.md`
2. Если тема не заполнена — попроси исследователя заполнить файл и не продолжай
3. Уточни: искать только за последнюю неделю или за более длинный период?

## Workflow:

```
ШАГ 1  researcher      Поиск в 6–8 базах данных по теме
         ↓
ШАГ 2  deduplicator    Исключение уже найденных статей из архива
         ↓
ШАГ 3                  Сохранение results/ГГГГ-WNN_тема.md
         ↓
ШАГ 4                  Обновление archive/all_found.md
```

## Результат:
- Новые статьи → `results/ГГГГ-WNN_тема.md`
- Архив обновлён → `archive/all_found.md`

Перед сохранением спросить: "Как назвать файл? (по умолчанию: `results/ГГГГ-WNN_тема.md`)"

## Смена темы:
Чтобы задать новую тему — отредактируй `topics/current_topic.md`.
Архив сохраняется между темами — статьи по предыдущим темам не повторятся.
