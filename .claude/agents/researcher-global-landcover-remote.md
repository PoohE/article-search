## Алгоритм работы

### Шаг 1 — Читай тему
Читай `topics/topic1_global_landcover.md`. Извлеки ключевые слова на английском и русском.

### Шаг 2 — Определи глубину поиска
Читай `archive/all_found_topic1.md`.
- Если DOI в архиве **нет** → ищи за последние **12 месяцев** (первый запуск)
- Если DOI есть → ищи за последний **1 месяц**

### Шаг 3 — Строй поисковые запросы

**Английские запросы:**
- Широкий: `global land cover remote sensing satellite classification`
- Узкий 1: `"global land cover" product validation accuracy`
- Узкий 2: `land cover mapping machine learning Sentinel Landsat`
- Узкий 3: `ESA WorldCover GlobeLand30 FROM-GLC comparison`

**Русские запросы:**
- `глобальные покрытия дистанционное зондирование классификация земного покрова`

### Шаг 4 — Поиск в базах данных

**Англоязычные источники:**
1. **Google Scholar** — `https://scholar.google.com/scholar?q=ЗАПРОС&as_ylo=ГОД`
2. **Semantic Scholar** — `https://api.semanticscholar.org/graph/v1/paper/search?query=ЗАПРОС&fields=title,authors,year,journal,externalIds,abstract`
3. **EarthArXiv** — препринты геонаук
4. **arXiv** — разделы cs.CV, eess.IV

**Русскоязычные источники:**
5. **КиберЛенинка** — `https://cyberleninka.ru/search?q=ЗАПРОС`
6. **eLIBRARY.ru** — поиск по русским ключевым словам
7. **Google Scholar** — русские запросы

Для каждой статьи фиксируй: DOI, авторы, год, название, журнал/квартиль, аннотация (2–3 предложения на русском), язык.

### Шаг 5 — Фильтрация качества

**Англоязычные — исключай если:** нет DOI, нет аффилиации, хищнический журнал (beallslist.net).
**Русскоязычные — принимай если:** есть в eLIBRARY / КиберЛенинке, есть журнал и авторы.

### Шаг 6 — Дедупликация
Читай `archive/all_found_topic1.md`. Исключи статьи чьи DOI уже есть в архиве.
Если DOI нет — проверяй по первым 8 словам названия.

### Шаг 7 — Формируй отчёт (формат ниже)

### Шаг 8 — Сохраняй результаты
1. Сохрани отчёт: `results/topic1_global_landcover/ГГГГ-MM_global-landcover.md`
2. Добавь DOI новых статей в `archive/all_found_topic1.md`

### Шаг 9 — Коммит и push
Выполни в bash:
```bash
git config user.email "claude-agent@anthropic.com"
git config user.name "Claude Agent"
git add results/topic1_global_landcover/ archive/all_found_topic1.md
git commit -m "Автопоиск тема 1 (глобальные покрытия): $(date +'%Y-%m')"
git push
```
